# GusevaK
Random Task Generator
Random Task Generator — это кроссплатформенное приложение, которое помогает выбрать случайную задачу из предопределённого списка. Идеально подходит для поиска вдохновения, борьбы с прокрастинацией или планирования дня.

Проект реализован на Python с использованием библиотеки Tkinter.

📦 Структура проекта
Файл	Описание
main.py	Основной исполняемый файл с логикой и графическим интерфейсом.
tasks.json	Список предопределённых задач и их категорий (учёба, спорт, работа).
history.json	Файл, в котором хранится история сгенерированных задач (создаётся автоматически).
.gitignore	Список файлов и папок, которые Git должен игнорировать.
README.md	Описание проекта, инструкции и информация об авторе.
⚙️ Технологии
Язык: Python 3.13.
Интерфейс: Tkinter (стандартная библиотека).
Данные: JSON.

✨ Возможности приложения
Генерация задачи: Нажмите кнопку, чтобы получить случайную задачу.
Фильтрация: Выберите категорию («Все», «Учёба», «Спорт», «Работа») для получения задач нужного типа.
История: Все сгенерированные задачи сохраняются в списке внизу окна.
Сохранение: История сохраняется в файл history.json и загружается при следующем запуске.
📝 Примеры использования (тесты)
Тест генерации: Запустите приложение и нажмите кнопку. В поле должна появиться задача.
Тест фильтрации: Выберите «Спорт» и нажмите кнопку. Задача должна относиться только к спорту.
Тест сохранения: Сгенерируйте задачу, закройте и откройте приложение снова — история должна сохраниться.
Тест пустого списка: Если в выбранной категории нет задач, приложение покажет предупреждение.
